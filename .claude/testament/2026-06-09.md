## Process gotcha worth remembering

My round-1 testament, staged-but-uncommitted, BLOCKED `git merge origin/main` with
"Your local changes to the following files would be overwritten by merge." The file isn't
on main — git just refuses to merge over a dirty index entry. The SC unstaged it and
started the merge. Lesson: if another merge may follow, do not leave the testament staged
between phases. Either commit it or leave it untracked. A staged-uncommitted file is the
one thing that silently blocks the next merge.
