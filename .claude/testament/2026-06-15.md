# 21:13 — beta.9 release (Phase 2 Courier)

This is the `1.0.0-beta.9` lockstep release. Five packages ship together. Phase 3 creates the GitHub releases that trigger npm publish.

## The release set

All five bumped to `1.0.0-beta.9`, in dependency order for Phase 3:

1. `claude-core` — `packages/claude-core`
2. `claude-sdk` — `packages/claude-sdk`
3. `claude-sdk-tools` — `packages/claude-sdk-tools`
4. `mcp-exec` — `packages/mcp-exec`
5. `claude-sdk-cli` — `apps/claude-sdk-cli`

`mcp-exec` is a transitive bump only — zero source changes since `beta.7`, and its `changes.jsonl` produced no new diff. Its `CHANGELOG.md` was not updated. Only its `package.json` moved to `beta.9`. When Phase 3 extracts release notes from mcp-exec's `[Unreleased]` section, expect it to be empty — that is correct.

All four other changelogs have entries under `[Unreleased]` only. No release markers — pre-release convention.
