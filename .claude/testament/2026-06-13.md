## What exists (ground truth before this cast)

The family fallback logic in `getFamilyDefault`:
```ts
return /^claude-(opus|sonnet)-/.test(modelId) ? 1_000_000 : 200_000;
```
`fable` is not in that regex, so any unmapped fable id (including `claude-fable-5`) currently falls back to 200k. The Builder needs to add `fable` to the regex (or equivalent) to satisfy requirements 2 and 3.
