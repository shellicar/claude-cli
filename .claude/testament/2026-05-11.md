## Structural decisions

**`overrides` object**: `let modelOverride` is extracted before `main()`. Inside `main()`,
wrap it as `const overrides = { model: modelOverride }` so issue #309 can mutate the slot
from a command-mode callback without touching `mapConfig`. The object goes just before
`const mapConfig = ...`.

**`buildInitialInput`**: Renamed from `buildFileInput`, gains a `text: string` parameter.
When `filePath` is null, returns immediately. When present, passes `text` through to
`buildSubmitText` — same call shape as the interactive path. No duplication of attachment
serialisation.

**`setSessionName` on `StatusState`**: State just stores it. The renderer decides whether
to show cwd or name. Clean separation.

**`BOLD_WHITE`**: Already in `@shellicar/claude-core/ansi`. Just add it to the import in
`renderStatus.ts`.
