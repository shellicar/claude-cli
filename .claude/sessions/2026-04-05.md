# Session 2026-04-05

## What was done

Extracted display/terminal utilities from `claude-cli` into `claude-core` so they can be reused by `claude-sdk-cli` and other consumers.

### New files in `packages/claude-core/src/`

- `sanitise.ts` - `sanitiseLoneSurrogates`, `sanitiseZwj`
- `reflow.ts` - `LineSegment`, `computeLineSegments`, `rewrapFromSegments`, `wrapLine`

- `status-line.ts` - `StatusLineBuilder`
- `viewport.ts` - `Viewport`, `ViewportResult`

- `test/viewport.spec.ts` - updated imports
